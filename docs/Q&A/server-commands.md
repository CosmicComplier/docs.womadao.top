## 个别指令说明
### /co lookup
这个指令可以查看周围方块的变化  
举例： /co l r:5 t:1d  
查看以自己为中心，5个方块为半径区域内，1天之中的方块变化  
可查看[co 使用说明](../Other/pluginsDocs/co-cha-jian-zhi-ling-shuo-ming)  
更多详情可以去看维基百科[wiki-coreprotect](https://mineplugin.org/CoreProtect)
